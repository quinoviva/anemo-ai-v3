================================================================================
                        TECHNICAL SUMMARY
================================================================================

Method: Brace-counting parser
  1. Iterate through file character-by-character
  2. Count: +1 for '{', -1 for '}'
  3. When count reaches 0 and we're on '}', we've found the end
  4. Extract everything up to that point
  5. Validate with json.loads()
  6. Write back cleaned content

Why this works:
  - JSON is properly nested, so counting braces is reliable
  - First complete JSON object always ends when depth returns to 0
  - Any garbage after that point can be safely discarded
  - Much faster and more reliable than regex or parsing heuristics

Complexity: O(n) time, O(1) space (one pass, no memory overhead)
